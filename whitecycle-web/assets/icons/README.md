# 图标资源说明

本文件夹用于存放应用所需的各种尺寸图标。

## 必需图标尺寸

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## 图标设计规范

- **格式**: PNG（支持透明度）
- **风格**: 简洁、现代
- **主色调**: 紫色渐变 (#6366f1 → #8b5cf6 → #a855f7)
- **内容**: 建议使用"白循"首字母或心形图案

## 图标生成工具

推荐使用以下工具生成多尺寸图标：

1. [Figma](https://www.figma.com/) - 在线设计工具
2. [Icon Generator](https://www.icon-generator.org/) - 批量生成
3. [RealFaviconGenerator](https://realfavicongenerator.net/) - PWA 图标生成

## 临时方案

在正式图标设计完成前，可以使用以下占位图：

```html
<!-- 使用 emoji 或文字作为临时图标 -->
<div style="font-size: 48px;">💖</div>
```
