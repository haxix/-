### 2. ✅ 将浏览器原生 confirm 改为自定义模态框

**问题**: 
- 使用浏览器原生 `confirm()` 弹窗
- 用户无法修改时间
- 样式不统一

**解决方案**:
- ✅ 使用自定义模态框
- ✅ 支持用户点击修改时长

- ✅ 样式与应用整体风格一致

## 🎨 新的时间确认流程

### 步骤 1: 点击"结束记录"
```
用户点击结束记录按钮
↓
显示自定义确认模态框
```

### 步骤 2: 确认/修改时长
```
┌─────────────────────────────────┐
│  ⏰ 时间有点长哦          ✕     │
├─────────────────────────────────┤
│                                 │
│  ⚠️ 温馨提示：                   │
│  要注意适当休息，                │
│  保持健康的生活方式！            │
│                                 │
│  记录时长                        │
│  ┌──────────────┐               │
│  │    35    │ 分钟  │ ← 可点击修改 │
│  └──────────────┘               │
│  💡 点击数字可以修改时长          │
│                                 │
│  [-5 分钟] [-1 分钟] [+1 分钟] [+5 分钟] │
│                                 │
│  [  取消  ]  [✅ 确认保存]      │
└─────────────────────────────────┘
```

### 步骤 3: 保存记录
```
用户点击"确认保存"
↓
如果修改了时长 → 更新记录数据
↓
关闭模态框
↓
显示庆祝和鼓励语
```

## 🔧 技术实现

### 1. 模态框创建

```javascript
function showTimeConfirmationModal(durationMinutes, onConfirm) {
    const modal = document.createElement('div');
    modal.className = 'modal-overlay active';
    
    modal.innerHTML = `
        <div class="modal modal-center">
            <!-- 标题和关闭按钮 -->
            <div class="modal-header">
                <h3>⏰ 时间有点长哦</h3>
                <button class="modal-close">✕</button>
            </div>
            
            <!-- 内容区 -->
            <div class="modal-content">
                <!-- 警告提示 -->
                <!-- 时长输入框 -->
                <!-- 快捷调整按钮 -->
                <!-- 确认/取消按钮 -->
            </div>
        </div>
    `;
    
    document.body.appendChild(modal);
}
```

### 2. 时长修改

```javascript
// 输入框
<input type="number" id="confirm-duration" 
       value="35" 
       min="0" max="999"
       onfocus="this.select()">

// 快捷按钮
<button onclick="document.getElementById('confirm-duration').value = 30">
    -5 分钟
</button>
```

### 3. 确认保存

```javascript
window.confirmWithDuration = function(btn, originalDuration) {
    const modal = btn.closest('.modal-overlay');
    const inputDuration = parseInt(document.getElementById('confirm-duration').value);
    
    // 如果用户修改了时长，更新记录
    if (currentRecording && inputDuration !== originalDuration) {
        currentRecording.actualDuration = inputDuration * 60;
        currentRecording.endTime = currentRecording.startTime + (inputDuration * 60 * 1000);
    }
    
    modal.remove();
    
    // 执行回调保存记录
    if (window.timeConfirmCallback) {
        window.timeConfirmCallback();
    }
};
```
