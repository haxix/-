# 密码保护功能说明

## 功能概述

白循 WhiteCycle 现已支持密码保护功能，使用 **SHA-256 哈希加密**和**加盐技术**保护您的隐私数据。

## 安全特性

### 🔐 加密方式
- **SHA-256 哈希算法**：密码经过不可逆的哈希变换，即使数据被窃取也无法还原
- **随机盐值**：每个密码都使用独特的随机盐值，防止彩虹表攻击
- **前端加密**：密码在浏览器本地完成加密，不会通过网络传输

### 💾 数据存储
- **LocalStorage**：加密后的密码哈希值存储在浏览器本地
- **SessionStorage**：登录状态仅在当前会话有效，关闭浏览器后自动登出

## 使用指南

### 启用密码保护

1. 进入「设置」页面
2. 点击「🔐 密码保护」
3. 输入账户名（至少 1 个字符）
4. 输入密码（至少 6 位）
5. 确认密码
6. 点击「启用密码保护」

### 登录

启用密码后，每次访问应用时都会看到登录界面：

1. 输入账户名
2. 输入密码
3. 点击「登录」

### 修改密码

1. 进入「设置」>「密码保护」
2. 点击「修改密码」
3. 输入当前密码
4. 输入新密码（至少 6 位）
5. 确认新密码
6. 点击「确认修改」

### 关闭密码保护

1. 进入「设置」>「密码保护」
2. 点击「关闭密码保护」
3. 确认关闭

### 忘记密码

如果忘记密码，可以选择「清除数据继续」：

⚠️ **警告**：此操作会清除所有记录和数据，应用将恢复到初始状态！

## 技术实现

### 密码加密流程

```javascript
// 1. 生成随机盐值（16 字节）
const salt = Utils.generateSalt();

// 2. 加盐哈希密码
const hashedPassword = await Utils.hashPasswordWithSalt(password, salt);

// 3. 存储盐值和哈希值
localStorage.setItem('whitecycle_auth_v1', JSON.stringify({
    username: username,
    hashedPassword: hashedPassword,
    salt: salt,
    enabled: true,
    createdAt: Date.now()
}));
```

### 密码验证流程

```javascript
// 1. 获取存储的盐值
const auth = JSON.parse(localStorage.getItem('whitecycle_auth_v1'));

// 2. 使用相同盐值哈希输入的密码
const hashedPassword = await Utils.hashPasswordWithSalt(inputPassword, auth.salt);

// 3. 比较哈希值
if (hashedPassword === auth.hashedPassword) {
    // 登录成功
    sessionStorage.setItem('whitecycle_logged_in_v1', 'true');
}
```

## 安全建议

1. **使用强密码**：建议包含大小写字母、数字和特殊字符
2. **定期修改密码**：提高安全性
3. **不要分享密码**：保护您的隐私
4. **备份数据**：定期导出数据以防忘记密码

## 注意事项

- ✅ 密码保护仅在当前浏览器有效
- ✅ 清除浏览器缓存会删除密码信息
- ✅ 关闭浏览器后需要重新登录
- ⚠️ 忘记密码无法找回，只能清除数据

## 版本信息

- **功能版本**：v1.0.0
- **加密算法**：SHA-256（Web Crypto API）
- **盐值长度**：16 字节（128 位）
