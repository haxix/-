## 📋 更新内容

### 1. 🗂️ 历史记录智能分组

**问题**: 记录多了看起来杂乱无章

**解决方案**:
- ✅ 按月份自动分组
- ✅ 每月显示摘要卡片（次数、总时长、平均时长）
- ✅ 点击摘要卡片可展开/收起该月记录
- ✅ 日期格式简化（`1/15 一 14:30`）
- ✅ 标签精简显示（`15min` 而不是 `15 分钟`）

**效果**:
```
📅 2024 年 1 月
┌─────────────────────────────────┐
│ 15 次记录 · 平均 12 分钟          │
│ 次数：15  总分钟：180            │
└─────────────────────────────────┘

1/20 六 15:30
⏱️ 15min 🧘

1/19 五 20:15
⏱️ 12min 💑
```

### 3. 📊 统计点击显示详情

**功能**: 点击任意统计卡片，显示详细数据

**支持**:
- ✅ 总记录 - 查看所有记录
- ✅ 本周 - 查看本周记录
- ✅ 本月 - 查看本月记录  
- ✅ 平均时长 - 查看平均数据

**详情模态框内容**:
- 大数字显示
- 文字说明
- 最近 10 条记录列表
- 滚动加载更多提示

**使用方式**:
```
点击统计卡片 → 弹出详情模态框 → 查看明细 → 关闭
```

---

### 4. ⏰ 智能时间确认

**功能**: 记录结束时，根据时长智能提示确认

**规则**:
- ⚠️ **超过 30 分钟**: 提醒注意休息
- ⚠️ **少于 1 分钟**: 确认是否保存
- ✅ **其他时长**: 直接保存

**对话框内容**:
```
本次记录时长为 35 分钟

时间有点长哦，要注意适当休息，
保持健康的生活方式！

确定要保存吗？

[取消] [确定]
```

## 🎯 使用指南

