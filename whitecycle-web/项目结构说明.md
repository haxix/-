## 🎯 核心文件说明

### 1. index.html
应用的主入口文件，包含所有页面结构和组件引用。

### 2. js/app.js
核心业务逻辑，包含：
- 页面管理
- 记录管理
- 数据统计
- 成就系统
- 智能建议

### 3. css/components.css
所有 UI 组件的样式，包括：
- Header（头部）
- Card（卡片）
- Button（按钮）
- Form（表单）
- Modal（模态框）
- Toast（通知）
- Calendar（日历）
- Chart（图表）
- Stats（统计）
- Settings（设置）

### 4. config/manifest.json
PWA 应用清单，使应用可以：
- 安装到桌面
- 离线使用
- 全屏显示

## 📝 开发建议

### 新增功能
1. 在 `app.js` 中添加业务逻辑
2. 在 `css/components.css` 中添加样式
3. 在 `index.html` 中添加 HTML 结构
4. 复杂功能可拆分为独立的服务或组件

### 修改样式
1. 颜色、字体、间距等 → 修改 `variables.css`
2. 组件样式 → 修改 `components.css`
3. 布局调整 → 修改 `layouts.css`
4. 动画效果 → 修改 `animations.css`

### 添加新页面
1. 在 `pages/` 目录下创建新文件夹
2. 创建 HTML 文件
3. 在 `app.js` 中添加路由逻辑
4. 在底部导航中添加导航项
