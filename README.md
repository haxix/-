适度放松，保持身心平衡
一、项目简介
「循白」是一款面向个人的私密健康管理工具，专注于记录、追踪和分析个人生活中的健康行为，帮助用户建立规律、健康的生活节奏，关注身心状态的平衡与变化。项目以简约干净的视觉设计为核心，兼顾易用性与隐私保护，为用户提供一个无评判、安全的记录空间。
二、核心功能亮点
✨ 多场景行为记录
支持「独处时光 / 二人世界 / 导师指导」等多种参与方式，可自定义工具使用场景，完整保存每一次记录的细节
表单设计简洁友好，一次记录即可完成所有信息填写，操作零门槛
📅 历史回溯与数据可视化
日历视图：直观展示记录时间线，随时回顾过往状态变化
统计看板：可视化数据图表，呈现行为频率、趋势与状态变化，辅助健康管理
支持按日 / 周 / 月维度查看数据，帮助用户了解自身习惯与变化
🔐 全方位隐私保护
本地加密存储：所有数据仅存储在用户本地设备中，使用 SHA-256 加密，不经过任何第三方服务器
一键设锁功能：右下角悬浮锁形按钮，点击即可快速锁定页面，防止他人误触查看，隐私安全一键守护
密码保护：支持设置独立访问密码，进一步保护记录数据
数据导入 / 导出：支持 JSON 格式数据导出与导入，用户可完全掌控自己的记录数据
🎨 个性化体验优化
亮色 / 暗黑双模式：支持一键切换界面主题，适配不同使用场景与光线环境
每日提醒：可开启健康提醒，帮助用户养成规律记录的习惯
响应式设计：适配电脑 / 平板等多种设备，随时随地都能轻松使用
三、技术栈与实现
前端：HTML + CSS + JavaScript，原生实现，无复杂依赖
数据存储：LocalStorage 本地加密存储，数据仅保存在用户设备
部署：GitHub Pages 免费静态托管，稳定访问
安全机制：SHA-256 加密算法 + 一键设锁 + 独立密码三重保护
四、使用指南
开始记录：打开网站，进入「记录」页面，选择参与方式，填写相关信息，提交即可完成记录
查看历史：切换到「历史」页面，通过日历视图查看过往所有记录
数据分析：进入「统计」页面，查看行为频率、趋势等可视化数据
隐私与设置：在「设置」中开启密码保护、切换主题、导出 / 导入数据；也可点击右下角「一键设锁」按钮快速锁定页面
五、项目设计理念
在快节奏的生活中，很多人忽略了个人健康的重要性。「循白」的设计初衷，是为用户提供一个私密、温和、无评判的记录空间：
界面设计以简约干净为主，避免过度设计带来的压迫感
隐私保护贯穿全程，从本地存储到一键设锁，让用户安心记录
功能聚焦实用，不做多余的社交化设计，回归个人健康管理的本质
六、隐私与开源说明
所有数据仅存储在用户本地设备中，不会上传至任何服务器
项目开源仅用于学习与个人使用，不收集任何用户信息
